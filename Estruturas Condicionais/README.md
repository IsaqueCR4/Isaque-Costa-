 # Sistema de Verificação de Idade

Este programa verifica se uma pessoa pode entrar em um evento de acordo com sua idade.

-Como funciona
O usuário informa a idade e o programa mostra uma das opções:
- Menores de 12 anos: entrada não permitida
- Entre 12 e 17 anos: somente com responsável
- 18 anos ou mais: entrada liberada

- Como executar
1. Abra o arquivo no Python
2. Execute o programa
3. Digite sua idade quando solicitado

-Exemplo
Entrada:
Digite sua idade: 20

Saída:
Entrada liberada! Aproveite o evento.
Obrigado por utilizar o sistema de verificação!
